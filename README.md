Repository containing the data and scripts used to map reef research in
the article of Osmar Luiz (Charles Darwin University, Australia).

Description of the files in this folder:

**Root**  
\|  
\|- **data**: raw datasets  
\| ——- *capitals.csv*: coordinates of capitals. Collected by ALLuza  
\| ——- *HistoricalData_60s_90s*: old papers missing DOIs  
\| ——- *BaseLines_2*: Brazilian Exclusive Economic Zone  
\| ——- *Basic_Data*: Basic Data from topic modeling (topic modeling
output; another ReadMe file can be found inside this folder)  
\| ——- *magris_reef_map*: Shapefiles of Brazilian reef distribution
(From Magris et al. 2020, Diversity and Distributions)  
\| ——- *MPAs*: Shapefiles of Brazilian Marine Protected Areas (From CNUC
2021)  
\| ——- *South_America*: Shapefile of south America  
\|  
\|- **docs**: presentation and meet record  
\|  
\|- **output**: raw plots  
\|  
\|- **1-Rscript_HTML**: folder containing R codes for processing
reviewed data:  
\|  
\| ——- *packages.R*: required packages  
\| ——- *retLongLatFunction.R*: regular expression to extract text from
html files (by Dr. Cristian Dambros, UFSM)  
\| ——- *extract_text_organize_data.R*: extract and organize data (call
the other two ‘.R’ files)

\|- **1-processed_data_HTML**: processed data after running the codes in
**1-Rscript_HTML**  
\|  
\| ——- *saved_text.RData*: extracted text (saved because it needs to be
extracted in the university web)  
\| ——- *spatial_data_topic_modeling.RData*: data organized for mapping
(extracted coordinates)  
\| ——- *spatial_data_topic_modeling.csv*: data organized for mapping
(extracted coordinates – file modified by O. Luiz to include coordinates
not extracted using the function “retLongLatFunction.R”)  
\| ——- *na_dois.csv*: papers with missing DOI  
\| ——- *get_missing_dois.csv*: get missing dois, same as in the data  
\| ——- *resu_dxdoi.RData*: coordinates extracted (originally not in
decimal degrees)  
\| ——- *findtLongLat_dxdoi.RData*: coordinates extracted (originally in
decimal degrees)

\|- **2-Rscript_mapping**: folder containing R codes for mapping and
producing the map in the main text:  
\|  
\| ——- *packages.R*: required packages  
\| ——- *mapping_of_papers.R*: codes used to create the spatial grid and
map research

\|- **2-processed_data_manual_coord_extraction**: folder containing
processed data after the addition of coordinates not captured by the
function “retLongLatFunction.R”:  
\|  
\| ——- *spatial_data_topic_modeling_3.csv*: dataset updated by Osmar
Luiz in 2023 (updated version of the file ‘spatial_data_topic_modeling’
generated by the function of HTML coordinate extraction)  
\| ——- *M2.csv*: topic modeling output  
\| ——- *topic_names.csv*: topic names  
\| ——- *topic_names.xlsx*: topic names
